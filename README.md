## Getting Started



### Prerequisites
Need to have python installed
To check go to terminal and type
* terminal
  ```terminal
  python --version
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/therealabdi2/LeetcodeQuestions
   ```
2. Install required packages
   ```sh
   pip install -r requirements.txt
   ```
   and you are good to go
